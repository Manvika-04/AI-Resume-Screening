# AI Resume Screening System

An AI-powered ATS Resume Screening web application built using Python and Flask.  
The system analyzes resumes against job descriptions and calculates ATS compatibility scores using Natural Language Processing (NLP).

---

# Features

- Upload multiple resumes
- ATS score calculation
- Resume and Job Description matching
- NLP-based text similarity
- Clean modern UI
- PDF resume support
- Recruiter-style screening system

---

# Technologies Used

- Python
- Flask
- HTML
- CSS
- Scikit-learn
- NLP
- TF-IDF Vectorization
- Cosine Similarity
